วิธีลง Mod ภาษาไทย



ใครที่คิดว่า DDLC ลงม็อดยาก คิดใหม่ได้เลย เพราะมันง่ายมากๆ ใครๆก็ทำได้

ลิ้งค์ไฟล์ภาษา กดตรงนี้เลย


หลังจากที่โหลดเสร็จจากนั้นเปิด Folder game ชึ้นมา / บน Steam กด Manage > Browse local files






หลังจากนั้นไปที่ Folder ที่ชื่อว่า game






ตอนนี้ก็ลากไฟล์จาก mod ลงมาใส่ได้เลย แล้วกด แทนที่ทั้งหมด หรือ Replace allได้เลย
